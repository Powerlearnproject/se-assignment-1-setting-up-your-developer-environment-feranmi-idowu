3. Set Up Version Control System
 Objective: Install Git and configure it on your local machine. I already have a GitHub account for hosting my repositories. 

Steps:
a. Install Git:
    - https://git-scm.com/download/win.
    - Download and run the installer.
    - Follow the installation steps, keeping the default options.
. Configure Git:
    - Open a terminal Command Prompt.
    - Set your Git username and email:
      git config --global user.name "Your Name"
      git config --global user.email "your.email@example.com"

 6. Configure a Database (MySQL)
  Objective: Download and install MySQL database.

Steps:
1. Download MySQL: https://dev.mysql.com/downloads/windows/installer/5.7.html.
2. Install MySQL:

7. Explore Extensions and Plugins
 Objective: Explore available extensions, plugins, and add-ons for your chosen text editor or IDE to enhance functionality.

Steps:
a. Open Visual Studio Code**:
    - Go to the Extensions view by clicking the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl+Shift+X`.
b. Install Extensions:
    - Search for and install the following recommended extensions:
        - Python: For Python development.
        - GitLens: Supercharge the built-in Git capabilities.
        - Pylint: For Python linting.
        - Prettier: For code formatting.
